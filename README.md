## Environment
```
$ node -v 
v10.15.3
$ npm -v 
6.4.1
```

## How to start (create project)
VUE CLIをインストール
```
npm install -g @vue/cli
```

Bootstrap-VUEをインストール　＊--save-dev で package.jsonにも追加
```
npm install --save-dev bootstrap-vue
```

VUE プロジェクトを新規作成
```
vue create my-project
```
